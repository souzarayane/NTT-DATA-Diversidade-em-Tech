Ferramentas Utilizadas

Visual Studio Code - VSCode

É um editor de código fonte desenvolvido pela Microsoft, sendo open source e disponível nas plataformas Windows, MacOS e Linux. Ele é totalmente customizável, podendo mpdificar tipo de fontes, a cor do tema, dark ou light, teclas de atalhos e etc.

Navegadores

Os navegadores, ou browsers em inglês, são programas ou aplicativos que servem para abrir uma página da internet. Na hora de verificar como sua página HTML está ficando é preciso usar um browser da sua preferência. Os navegadores mais conhecidos são:

  • Chrome
  • Safari
  • Firefox
  • Edge
  • Opera

Ferramenta do Desenvolvedor

A ferramenta do desenvolvedor está presente em todos os navegadores e ajuda muito a verificar a estrutura da página web. É possível fazer manipulações na página como: mudar de cor de fundo, alterar fonte, tirar imagem entre outros. Tudo isso pode ser feito sem a necessidade de salvar as mudanças.

Estrutura Básica do HTML

Estrutura do Documento

A estrutura básica do HTML é dividida em 3 partes principais, que são:

  1. HTML - onde começa com html e termina com /html;
  2. HEAD - onde se encontra o cabeçalho da página;
  3. BODY - onde todo o conteúdo do site está.

Falando Sobre Tags

Existem vários tipos de tags diferentes no HTML, vopu citar algumas delas:

  • I - tag que deixa o texto em itálico
  • Strong - tag que deixa o texto em negrito
  • Input - é um campo, por exemplo do tipo texto

Início

Atributo das Tags

Atributos são usados dentro das tags HTML para modificar o comportamento do elemento. Exemplos:

  • Style - esse atributo pode mudar a cor, fonte, tamanho do elemento
  • Class - classe onde uma tag pode ser padronizada e reaproveitada em outras tags
  • Input type number - um campo que só aceita o tipo número
  • Input type color - um campo que aceita escolher cores
  • Input type text - um campo que só aceita o tipo texto
  • Img - tag que coloca imagem no documento HTML

Links

Quando se quer navegar de uma página a outra ou saltar para uma determinada parte da página, sem seguir uma ordem, se usa o link. O link pode ser um texto, figura ou uma foto.

O atributo que especifica onde o link vai ser aberto é o target. Esse atributo pode direcionar o link a abrir na mesma janela usando o target="_self", pode também apontar para outra janela usando o target="_blank".
